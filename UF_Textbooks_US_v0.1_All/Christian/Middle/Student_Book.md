## Unit 1: Better Than Baseline
### Why this matters
We learn to judge **actions**, not people. We compare an action to a **baseline** and ask if it improves things within agreed **caps** (harm, consent, fairness, honesty, bias/opacity, reversibility). If yes, it’s **Right (↑)**; if not, it’s **Wrong (↓)**. The perfect endpoints (Good*/Evil) are ideals, not today’s claims.

We measure **ReliefIndex (RI)**:

$$RI = \frac{H(\text{baseline}) - H(\text{action})}{\max(H(\text{baseline}),\varepsilon)}$$

- If RI ≥ 0 and caps pass → **Right (↑)**. If RI < 0 or any cap fails → **Wrong (↓)**.

**Practice:** For a baseline harm of 0.20 and action harm of 0.15, compute RI. Does it pass if the harm cap is 0.10 for low-stakes? Explain.
