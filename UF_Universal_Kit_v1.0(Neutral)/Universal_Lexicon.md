# UF Universal Lexicon (v1.0)
**Date:** 2025-08-06

This lexicon removes theological language and uses neutral, universal terms.

| Prior Term | Universal Term | Short Definition |
|---|---|---|
| Good* | **Aligned Limit** | Action state approaching maximum alignment (A→1) with capped/vanishing expected harm and all gates passing. |
| Evil | **Collapse Limit** | Action state approaching maximum misalignment (M→1) and/or persistent red‑line violations. |
| Right | **Improves vs Baseline (Pass)** | Expected outcomes better than baseline within caps; gates pass. |
| Wrong | **Degrades vs Baseline (Fail/Escalate)** | Expected outcomes worse than baseline or gates fail. |
| Heaven/Hell | **High‑Alignment / Misalignment Basins** | Stable regions of behavior that drive A up or down over time. |
| Return to Potential | **Reset to Live Superposition** | Pause + re‑aim + consent/fairness repair + next gate‑passing step. |
| Coherence | **Coherence Index** | Bounded index combining alignment, integrity, ethics, consent/fairness, non‑harm. |
| God (metaphor) | — (omit) | No mapping; UF remains descriptive/operational. |
