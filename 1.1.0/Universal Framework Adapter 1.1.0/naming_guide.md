# Naming Guide (Neutral & Human)

## Roles
- Author / Contributor / Editor
- Reviewer (logic/evidence)
- Maintainer (versions/profiles)
- Implementer (integrates VC/HL)
- User / Community

## Artifacts
- White Paper
- Policy Profile (JSON)
- Core Invariants (I1–I7)
- Verification Core (VC)
- Holographic Ledger (HL)
- Adapter Notes (mapping from principles to profile)

## Claims Language
- Proposed / Conjecture / Evidence suggests / We show
- Avoid prophetic framing (prophet, oracle, revelation)
- Attribution: “Composed by <Name> as a human contributor.”
