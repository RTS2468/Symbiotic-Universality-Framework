UF Companion v2.0 Package
===========================

This package contains the draft preprint, a JSON verification profile, and supporting files for the Universality Framework Companion v2.0.

Files included:
- UF_Companion_Preprint_v2.0.tex : LaTeX source for the technical preprint
- UF_Companion_Profile_v2.0.json : Example verification profile schema
- README.txt : This file

Usage:
1. Compile the .tex file with pdflatex to generate the preprint PDF.
2. The JSON file can be validated using adapter_universal.py or similar verification core tools.
