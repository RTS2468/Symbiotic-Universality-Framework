UF Companion v2.0 Fancy Whitepaper
=================================

This is the LaTeX source for the Fancy Whitepaper version of the Universality Framework Companion v2.0.

Compile instructions:
1. Place this .tex file into a LaTeX environment (e.g., Overleaf, TeXLive).
2. Compile with pdflatex or lualatex to produce UF_Companion_Fancy_Whitepaper_v2.0.pdf.

This Fancy Whitepaper complements the Preprint version by including quotes, metaphors, and story-driven examples of UF applied to companionship.
